Implementierung

Die USDZ-Datei enth\u00e4lt ein USD VariantSet namens materialVariant mit zwei Optionen. AR Quick Look erkennt das VariantSet automatisch und zeigt einen Picker im AR-Viewer.

Der Apple Pay Banner wird \u00fcber URL-Fragment-Parameter gesteuert:

<a rel="ar" href="cube_configurator.usdz #applePayButtonType=plain &checkoutTitle=3D%20Cube &checkoutSubtitle=Farbvarianten &price=%E2%82%AC9%2C99"> <img src="preview.png"> </a>

Event Listener (Apple Pay Tap)

Wenn der User auf den Apple Pay Button tippt, wird AR Quick Look geschlossen und ein Event gefeuert:

document.querySelector('a[rel="ar"]') .addEventListener('message', (e) => { if (e.data === '_apple_ar_quicklook_button_tapped') { window.location.href = '/checkout'; } });
